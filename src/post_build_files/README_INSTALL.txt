Instrukcja instalacji i uruchomienie programu MGservice:

1. Zainstaluj środowisko uruchomieniowe Node.js w wersji 12.18.4 z https://nodejs.org/download/release/v12.18.4/

2. Zresetuj komputer.

3. Uruchom "Wiersz polecenia" i sprawdź, czy Node.js działa - wpisz polecenie: "node -v". Wynikiem powinna być wersja Node.js, czyli np. "v12.18.4". Jeśli tak jest przejdź do pkt. 5.

4. Jeśli Node.js nie działa prawdopodobnie musisz dodać do zmiennej środowiskowej PATH ścieżkę "C:\Program Files\nodejs" i zrestartować komputer.

5. Uruchom "Wiersz polecenia" i zainstaluj serwer plików bazy danych json-server - wpisz polecenie: "npm install -g json-server"

6. Uruchom "Wiersz polecenia" i zainstaluj serwer plików statycznych angular-http-server - wpisz polecenie: "npm install -g angular-http-server" 

7. Utwórz skrót dla pliku run.cmd i umieść go na pulpicie. Możesz mu zmienić nazwę i ikonę.

8. Uruchom skrót programu MGservice. Aplikacja powinna się otworzyć w domyślnej przeglądarce interenetowej.
